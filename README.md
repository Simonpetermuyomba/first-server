# My First Server with GET Route

This project demonstrates a simple server using Node.js and Express that responds to GET requests.

## Getting Started

1. **Clone the repository**:
   ```bash
   git clone https://github.com/simonpetermuyomba/first-server.git
   cd first-server
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Run the server**:
   ```bash
   npm start
   ```

4. **Access the GET route**:
   Open `http://localhost:3000/` in your browser to see the response.
